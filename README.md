Car Control Project

This is part of the project to built a car control
for a KitCar.
PCBs are created using Fritzing and are home made.

It is split into 3 different PCBs and responsibilities:
1. Lights
2. Sensors
3. Indicators


Mainlights

The shield controls:

- Mainlight (beam)
- Diplight
- Backlights
- Foglight
- Parklights
